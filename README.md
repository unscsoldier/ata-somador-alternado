## Enunciado

Escreva um programa que lê dois números inteiros e calcula a soma dos números entre eles, alternando entre adição e subtração. Por exemplo, se os números forem 5 e 2, o cálculo seria: 3 - 4 + 5 = 4. Considere que os números são sempre inteiros e o primeiro será sempre maior que o segundo.

Exemplo:
Entrada:
5 2
Saída:
4
